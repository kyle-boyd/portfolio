# TOC icons (main page case studies)

Add two SVGs per project: **grayscale** (default) and **color** (hover/active).

| Project            | Grayscale file                 | Color file               |
|--------------------|--------------------------------|---------------------------|
| Syncrofy           | `syncrofy-grayscale.svg`       | `syncrofy-color.svg`      |
| Lighthouse         | `lighthouse-grayscale.svg`    | `lighthouse-color.svg`    |
| Acelab             | `acelab-grayscale.svg`        | `acelab-color.svg`       |
| Design System Work | `design-system-grayscale.svg` | `design-system-color.svg` |

Until these files exist, the TOC falls back to the project card image.
